# 안드로이드 레이아웃

## Pixel Density

- 1인치에 들어맞는 픽셀의 수
- 고밀도의 화면은 저밀도의 것보다 인치 당 더 많은 픽셀을 가진다.
- 버튼과 같은 UI 요소는 저밀도의 화면에서 더 크게 보이고, 고밀도의 화면에서 더 작게 보인다.

## DP(Density-independent pixels)

- 안드로이드에서 사용되는 독립적 단위 수치
- 어떠한 해상도에서도 같은 크기를 보여줌
- px = dp * (dpi / 160)

## SP(Scaleable pixels)

- 스케일에 따른 독립적 단위 수치
- 기본적으로 서체 크기에 사용됨
